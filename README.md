# OpScaler

**Build. Ship. Scale.**

OpScaler is a US-based technology company that helps startups and small businesses turn ideas into products. Founded by [Pavan Tikkani](https://linkedin.com/in/pktikkani) and [Karthik Sethupathy](https://linkedin.com/in/karthiksethupathy).

## Services

- **AI & Machine Learning** — Custom AI integrations, chatbots, RAG pipelines, intelligent automation
- **Web & Mobile Development** — React, Next.js, React Native apps from MVP to production
- **Cloud Infrastructure** — Right-sized AWS / GCP / Azure setups, cost-optimized
- **DevOps & Automation** — CI/CD, Docker, Kubernetes, monitoring, infrastructure as code

## Tech Stack

- [Next.js](https://nextjs.org) 16.2
- [React](https://react.dev) 19.2
- [Tailwind CSS](https://tailwindcss.com) 4.2
- [Framer Motion](https://motion.dev) 12
- TypeScript 5.8

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

Create a `.env.local` file for the contact form:

```
NEXT_PUBLIC_WEB3FORMS_KEY=your_key_here
```

Get a free key at [web3forms.com](https://web3forms.com).
